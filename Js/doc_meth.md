# 📜 Шпаргалка: JavaScript Document Object Model (DOM)

Объект `document` — это основной интерфейс для управления HTML-документом. Он позволяет находить, изменять, создавать и удалять элементы страницы.

---

## 🔍 1. Методы поиска элементов
Способы «зацепиться» за нужный тег в дереве.

| Метод | Что возвращает | Описание |
| :--- | :--- | :--- |
| `getElementById('id')` | 1 элемент | Поиск по уникальному атрибуту `id`. Самый быстрый. |
| `querySelector('selector')` | 1 элемент | Первый элемент, подходящий под CSS-селектор. |
| `querySelectorAll('sel')` | `NodeList` | Все элементы по селектору. Можно перебирать через `.forEach()`. |
| `getElementsByClassName('c')` | `HTMLCollection` | Все элементы с данным классом. "Живая" коллекция. |
| `getElementsByTagName('t')` | `HTMLCollection` | Все элементы с указанным тегом (например, `div`, `p`). |

---

## 🏗️ 2. Создание и удаление узлов
Динамическое изменение состава страницы.



### Создание
```javascript
const newElement = document.createElement('div'); // Создать тег
newElement.textContent = 'Привет, мир!';         // Добавить текст
newElement.classList.add('my-style');            // Добавить класс
```

## 🛠️ 3. Манипуляция и вставка узлов
Методы для перемещения элементов и изменения их положения в DOM-дереве.



| Метод | Описание | Пример |
| :--- | :--- | :--- |
| `append(...nodes)` | Вставляет узлы в **конец** родителя. | `parent.append(div);` |
| `prepend(...nodes)` | Вставляет узлы в **начало** родителя. | `parent.prepend(div);` |
| `before(...nodes)` | Вставляет узлы прямо **перед** элементом. | `el.before(newDiv);` |
| `after(...nodes)` | Вставляет узлы сразу **после** элемента. | `el.after(newDiv);` |
| `replaceWith(node)` | **Заменяет** текущий элемент на новый. | `el.replaceWith(newDiv);` |

### Пример перемещения элемента:
```javascript
const list = document.querySelector('ul');
const newItem = document.createElement('li');
newItem.textContent = 'Я в самом начале!';

list.prepend(newItem); // Элемент станет первым в списке
```
## 📋 4. Полезные свойства объекта `document`

Свойства позволяют получить быстрый доступ к ключевым элементам страницы и метаданным без использования методов поиска.



### Основные свойства доступа к узлам
| Свойство | Описание |
| :--- | :--- |
| `document.documentElement` | Ссылка на корневой элемент `<html>`. |
| `document.body` | Ссылка на элемент `<body>`. |
| `document.head` | Ссылка на элемент `<head>`. |
| `document.activeElement` | Элемент, который в данный момент находится в фокусе. |

### Информационные свойства
| Свойство | Описание |
| :--- | :--- |
| `document.title` | Текст в заголовке вкладки браузера. Можно перезаписывать. |
| `document.URL` | Полный адрес (URL) текущей страницы. |
| `document.characterSet` | Кодировка документа (например, `UTF-8`). |
| `document.readyState` | Состояние загрузки (`loading`, `interactive`, `complete`). |
| `document.cookie` | Строка со всеми куки-файлами, доступными для данного домена. |

---

### Примеры использования

**1. Динамическое изменение заголовка:**
```javascript
// Полезно для привлечения внимания пользователя
document.title = "⚠️ У вас новое сообщение!";
```
## ⚡ 5. События документа

Объект `document` позволяет отслеживать состояние жизненного цикла страницы и глобальные действия пользователя.



### События жизненного цикла
| Событие | Описание |
| :--- | :--- |
| `DOMContentLoaded` | Браузер полностью загрузил HTML и построил DOM-дерево. Картинки и стили ещё могут грузиться. |
| `readystatechange` | Отслеживает изменение состояния загрузки (`document.readyState`). |

### Глобальное отслеживание событий
Вы можете вешать обработчики прямо на `document`, чтобы ловить события во всей вкладке.

```javascript
// 1. Ожидание готовности DOM (рекомендуемый способ запуска скриптов)
document.addEventListener('DOMContentLoaded', () => {
    console.log('DOM готов, можно искать элементы!');
});

// 2. Делегирование событий
// Вместо того чтобы вешать клик на каждую кнопку, вешаем один на весь документ
document.addEventListener('click', (event) => {
    if (event.target.closest('.btn-delete')) {
        console.log('Нажата кнопка удаления!');
    }
});
