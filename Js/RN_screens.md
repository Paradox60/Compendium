# 1️⃣ Установка React Navigation

Для управления экранами в React Native почти всегда используют библиотеку React Navigation.

Установка базовых пакетов:
```bash
npm install @react-navigation/native
```

```bash
npm install react-native-screens react-native-safe-area-context
```
```bash
npm install @react-navigation/native-stack
```
	Если будешь на Expo, просто expo install вместо npm install.

# 2️⃣ Настройка навигации

Создадим папку src/screens/ и несколько экранов.

Структура:

src/
 └── screens/
      ├── HomeScreen.js
      ├── WordsScreen.js
      └── TrainingScreen.js
