

# Создание Flask в ручном режиме:
---

#### 1. Создать папку проекта project_folder

```bash
mkdir project_folder
```

#### 2. Создаем виртуальное окружение

```bash
python3 -m venv venv
```

#### 3. Активируем его

```bash
source venv/bin/activate  # для Linux/macOS
# .\venv\Scripts\activate  # для Windows
```

#### 4. Устанавливаем фласк

```bash
pip install flask
```
#### 5. Прописываем bash функцию на инициализацию проекта

```bash
start_flask_project
```


# Запуск 
---

```bash
python 3 app.py
```
