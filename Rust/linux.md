### Системные зависимости для работы winit


```bash
sudo apt update
```
### Установка зависимостей для winit

```bash
sudo apt install -y \
    libx11-dev \
    libx11-xcb-dev \
    libxcb1-dev \
    libxcb-randr0-dev \
    libxcb-shape0-dev \
    libxcb-xfixes0-dev \
    libxkbcommon-dev \
    libwayland-dev
```

# Ubuntu/Debian
```bash
sudo apt install libx11-dev libxrandr-dev libxcb-render0-dev libxcb-shape0-dev libxcb-xfixes0-dev libxcb1-dev libwayland-dev
```
