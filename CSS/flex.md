
# <center>Свойства display: flex;</center>
### **Хорошая статься на этот счет:**
https://tproger.ru/translations/how-css-flexbox-works

- ### **flex-direction:** 
	##### *row, column*
	*Отвечает за направление*
	<br>

- ### **justify-content:**
	##### *flex-start,flex-end,center,space-between,space-around*
	*Отвечае за выравнивание элементов по главной оси:*
	<br>

- ### **align-items:** 
	##### *flex-start,flex-end,center,stretch,baseline*
	*Работает с осью, перпендикулярной главной оси.*
	<br>

- #### ***flex-grow***
*Отвечает за то, как элемент будет расти, если в контейнере остаётся свободное место.*
	- ***0 → (по умолчанию) не растёт.***
	- ***1 → может занять всё свободное место.***
	- ***2, 3, ... → чем больше число, тем больше доля свободного пространства.***
<br>

- #### ***flex-shrink***
*Отвечает за то, как элемент будет сжиматься, если места мало.-*
	- ***1 (по умолчанию) → сжимается.***
	- ***0 → не сжимается.***
	- ***2, 3... → элемент будет ужиматься быстрее других.***
<br>

- #### ***flex-basis***
*Это начальный размер элемента (до распределения свободного места).
Можно задавать:*
	- ***auto → берётся ширина/высота из стилей.***
	- ***100px → фиксированная "база".***
	- ***0 → база равна нулю, и размер целиком зависит от flex-grow.***
