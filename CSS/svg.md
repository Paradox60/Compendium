# SVG (Scalable Vector Graphics)

**SVG** — это векторная графика, описанная кодом (XML). Вместо пикселей используются **фигуры, линии и кривые**, поэтому изображение остаётся чётким при любом масштабе.

---

## Основная идея

SVG — это не картинка, а **HTML‑подобная разметка**:

```html
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="#00bfff" />
</svg>
```

Браузер рисует фигуру по описанию.

---

## Преимущества SVG

* масштабируется без потери качества
* маленький размер файлов
* можно стилизовать через CSS
* можно управлять через JavaScript
* отлично подходит для иконок и UI

---

## Основные SVG‑элементы

```html
<rect />     <!-- прямоугольник -->
<circle />   <!-- круг -->
<ellipse />
<line />
<polygon />
<path />     <!-- сложные формы -->
```

Большинство иконок состоит из одного или нескольких `<path>`.

---

## Атрибуты формы

Пример круга:

```html
<circle cx="50" cy="50" r="40" fill="#00bfff" />
```

* `cx`, `cy` — центр
* `r` — радиус
* `fill` — заливка
* `stroke` — обводка

---

## SVG и CSS

```html
<svg class="icon" viewBox="0 0 24 24">
  <path d="..." />
</svg>
```

```css
.icon {
  width: 32px;
  height: 32px;
  fill: #00bfff;
}

.icon:hover {
  fill: #ffffff;
}
```

SVG можно стилизовать так же, как обычные HTML‑элементы.

---

## SVG и JavaScript

```js
document.querySelector('.icon')
  .addEventListener('click', () => {
    console.log('SVG clicked');
  });
```

SVG‑элементы являются частью DOM.

---

## `viewBox` — важнейшая часть

```html
<svg viewBox="0 0 24 24">
```

`viewBox` задаёт **внутреннюю систему координат** SVG и позволяет масштабировать изображение без искажений.

---

## Когда использовать SVG

✔ иконки интерфейса
✔ логотипы
✔ схемы
✔ интерактивные элементы
✔ анимации линий

❌ фотографии и сложные растровые изображения

---

## SVG vs PNG / JPG

| PNG / JPG            | SVG           |
| -------------------- | ------------- |
| пиксели              | векторы       |
| теряет качество      | всегда чёткий |
| тяжёлый              | лёгкий        |
| нельзя править кодом | можно         |

---

## SVG vs `clip-path`

|                    | SVG | clip-path |
| ------------------ | --- | --------- |
| Точная форма       | ✅   | ❌         |
| Кликабельная форма | ✅   | ❌         |
| Простота           | ❌   | ✅         |
| UI‑декор           | ⚡   | ⭐         |

---

## Краткий вывод

**SVG** — лучший инструмент для иконок, логотипов и точных форм. Он отлично сочетается с CSS и JavaScript и идеально подходит для современных интерфейсов.
