# CSS `clip-path`

`clip-path` — это CSS‑свойство, которое **обрезает видимую область элемента по заданной форме**. Всё, что находится за пределами формы, становится невидимым.

> `clip-path` не рисует фигуру, а работает как маска.

---

## Базовая идея

```css
div {
  clip-path: circle(50%);
}
```

Элемент остаётся прямоугольным, но **отображается как круг**.

---

## Основные формы

### `circle()` — круг

```css
clip-path: circle(40% at 50% 50%);
```

* первый параметр — радиус
* `at X Y` — положение центра

---

### `ellipse()` — овал

```css
clip-path: ellipse(60% 40% at 50% 50%);
```

---

### `inset()` — обрезка прямоугольника

```css
clip-path: inset(10px 20px round 12px);
```

Похоже на `padding + border-radius`, но без влияния на layout.

---

### `polygon()` — произвольная форма

```css
clip-path: polygon(
  50% 0%,
  100% 25%,
  100% 75%,
  50% 100%,
  0% 75%,
  0% 25%
);
```

Используется для:

* стрелок
* шестиугольников
* техно‑панелей

---

## Как работают координаты

* `0% 0%` — левый верхний угол
* `100% 0%` — правый верхний
* `100% 100%` — правый нижний
* `0% 100%` — левый нижний

Проценты считаются **от размеров элемента**.

---

## Важные особенности

* `clip-path` **не влияет на размеры и поток документа**
* кликабельная зона остаётся прямоугольной
* содержимое внутри тоже обрезается

---

## Анимация

```css
.card {
  transition: clip-path 0.3s ease;
}

.card:hover {
  clip-path: inset(6px round 10px);
}
```

---

## Когда использовать

✔ нестандартные формы UI
✔ декоративные панели
✔ hover‑эффекты

❌ если нужна точная кликабельная форма (лучше SVG)

`clip-path` — мощный инструмент для создания форм **без изображений и SVG**, идеально подходит для современных интерфейсов и декоративных эффектов.
